PyYama

The replacement for the Web UI in multi-room audio devices

Screenshot of the user interface.

PyYama is a Python 3 and PyQt 5 based GUI, which can control certain pieces of multi-room audio devices designed and manufactured by a multinational Japanese company. This project is not sanctioned by said company. Neither this page nor PyYama itself make no references to this company, its name or its registered trademarks.

You can download the latest version of PyYama from GitHub.
