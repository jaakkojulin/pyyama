PyYama

The replacement for the Web UI in certain streaming music players

Screenshot of the user interface.

PyYama is a Python 3 and PyQt 5 based GUI, which can control certain pieces of home entertainment systems created by a multinational Japanese company. Since this project is not sanctioned by said company this page not PyYama makes no references to this company, its name nor its registered trademarks.

The application is developed by Jaakko Julin and will be released soon under the GPL licence.

The primary advantages of using PyYama are:

  • More bugs
  • Less fun

Requirements:

  • Relatively modern Windows, OSX or Linux
  • Python 3
  • PyQt5
  • Requests
